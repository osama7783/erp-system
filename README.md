# mikatakno ERP System

## Overview
MechaTechno ERP is a lightweight enterprise resource planning system designed to simulate real-world business workflows between administrators, employees, and customers.

The system manages the full lifecycle of purchase requests, from submission and pricing to approval and operational processing.

---

## Core Workflow
1. Customer submits a purchase request  
2. Employee receives and reviews the request  
3. Employee provides pricing (quotation)  
4. Customer reviews and approves the quotation  
5. Process continues to operations (inventory, shipping, etc.)

---

## Main Modules

### User Roles
- Admin
- Employees
- Customers

### Features
- Purchase request management  
- Quotation and pricing system  
- Shipment management  
- Inventory (warehouse) management  
- Customer management  
- Financial operations  
- Financial transfer tracking  
- Factory and supplier management  

---

## Tech Stack
- Framework: Next.js  
- Frontend: React.js  
- Language: JavaScript  
- Backend & Database: Supabase  
- Styling: Tailwind CSS  
- Data Handling: JSON  
- File Processing: XLSX  

---

## Architecture Highlights
- Role-based access system  
- Full workflow between multiple user types  
- API-driven communication  
- Scalable and modular structure  
- Clean UI with modern styling
